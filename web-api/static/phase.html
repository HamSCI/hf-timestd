Carrier Phase vs Time

Unwrapped phase tracks ionospheric path length. Ramps = Doppler drift. Jumps = mode changes.

Doppler Shift vs Time

fD = -(1/2π) dφ/dt. Positive = ionosphere descending. Sunrise/sunset produce characteristic signatures.

Phase Scintillation Index (σφ)

Standard deviation of detrended phase over 60s windows. High σφ (> 0.3 rad) = ionospheric irregularities.

DC Carrier Phase (Unambiguous Channels)

CHU (3.33/7.85/14.67 MHz) and WWV (20/25 MHz): bare carrier phase, independent of tone detection. Cleanest ionospheric observable.
