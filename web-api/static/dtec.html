dTEC — Multi-Station Carrier-Phase TEC

Carrier-phase differential TEC rate and integrated TEC across all 17 propagation paths. Correlated signatures indicate Traveling Ionospheric Disturbances (TIDs).

Detection capability: Carrier-phase dTEC achieves ~6 mTECU/min noise floor (55 ticks/min, ~1 mrad phase noise at 20 dB SNR). TIDs produce 100–2000 mTECU perturbations → 17–330× above noise floor. Correlated signatures across WWV (SW), WWVH (W), CHU (NE), and BPM (W) paths constrain the TID wavevector direction and speed.

Note: Group-delay TEC (from timing residuals) is below the noise floor (~6.5 ms model error vs ~0.85 ms dispersion signal). Carrier-phase dTEC is the primary ionospheric product.
