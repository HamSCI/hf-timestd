Interpretation guide (after Griffin 2026): Rank-0 arrivals (dominant path) cluster near the model-predicted ToF. Rank-1+ arrivals (secondary paths) appear at longer delays — each cluster corresponds to a distinct propagation mode (1F, 2F, sporadic-E, sidescatter). The delay difference between rank-0 and rank-1 directly encodes the F2 virtual height. SNR separates modes: higher-hop paths are weaker due to additional D-layer transits.
